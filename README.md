# 🗺️ Türkiye Açık Hava Müzeleri ve Göbeklitepe Gezi Rehberi

Yazılım Mühendisliği web geliştirme çalışmaları kapsamında, modern web standartlarına uygun olarak geliştirilmiş kullanıcı dostu ve içerik odaklı bir blog arayüzü projesidir. 

Bu projede, tarihin sıfır noktası olarak kabul edilen **Göbeklitepe** başta olmak üzere Türkiye'nin önemli açık hava müzeleri ve antik kentleri dijital bir rehber konseptiyle listelenmiştir.

---

## 🚀 Kullanılan Teknolojiler

Projenin arayüz tasarımı ve responsive (mobil uyumlu) yapısı tamamen modern frontend araçları kullanılarak kurgulanmıştır:

* **HTML5:** Web sayfasının semantik (anlamsal) iskeletinin oluşturulması.
* **CSS3:** Temel görsel düzenlemeler ve yerleşimler.
* **Bootstrap 5:** Grid sistemi, hazır bileşenler (Navbar, Cards, List Groups, Badges) ve responsive tasarım altyapısı.

## 📌 Öne Çıkan Özellikler

* **Mobil Uyumlu Tasarım:** Bootstrap 5 grid sistemi sayesinde telefon, tablet ve masaüstü ekranlarda kusursuz görünüm.
* **İçerik Odaklı Sidebar (Yan Menü):** Kullanıcıların son gönderilere ve popüler etiketlere hızlıca ulaşabileceği dinamik görünüm.
* **Modern Tipografi:** Okunabilirliği yüksek başlıklar ve alıntı (`blockquote`) alanları.
